# Water-Engine
Play as Lara Croft in a new kind of adventure. A village dear to one of her friends is on the brink of collapse, its water supply unfinished and destroyed. Only you can restore it.

Getting Started :

1. Click on the "Releases" section (bottom right of this page)
2. Download the WATER_ENGINE.zip
3. Run PLAY.exe to launch the game
4. Load the only available level
5. Have fun !




Controls :

&#x09;Movement :


&#x09;	W / ↑ – Move forward

&#x09;	S / ↓ – Move backward

&#x09;	A / ← – Turn left

&#x09;	D / → – Turn right




&#x09;Actions :


&#x09;	Space – Jump, swim, roll

&#x09;	Ctrl – Action (grab, interact, climb, etc.)
